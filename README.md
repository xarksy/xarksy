### Hi there 👋
I am a self-taught developer with a strong passion for programming. Early on, I found repetitive tasks dull—until I discovered AutoIt with Selenium WebDriver a few years ago. The moment I successfully automated a browser action with a simple script, I was hooked. That experience sparked my deep interest in software development and problem-solving through code.

Since then, programming has become an engaging journey of continuous learning and improvement. I enjoy the challenge of building high-quality software and continuously expanding my technical skills. What started as writing simple scripts has grown into developing full-scale applications, making the process even more exciting.

I have successfully built Android applications using Kotlin, integrating Firebase services for features such as authentication and real-time data handling. On the backend side, I develop web applications using Django and design RESTful APIs to support scalable and maintainable systems. I also use Django as a core backend and am currently exploring microservices architecture to further enhance system flexibility and performance.

Driven by curiosity and a commitment to growth, I continue learning through platforms like Dicoding, where I deepen my knowledge and stay up to date with modern development practices.
